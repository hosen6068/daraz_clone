## 🎯 My Features (Remember Me & Cookies)

### How Remember Me Works:

1. **User Enables Remember Me During Login**
   - Secure token generated (32 bytes of random data)
   - Token hashed using SHA-256
   - Token stored in database with 30-day expiration
   - Cookie set for 30 days (httpOnly, secure)

2. **What Gets Saved**
   - Hashed token in database
   - User email in cookie (remember_email)
   - Secure token in cookie (remember_token)

3. **Next Visit (Auto-Login)**
   - System checks for remember_token cookie
   - Validates token against database
   - Checks if token hasn't expired
   - Auto-logs in user if valid
   - Shows saved email for manual login option

4. **Security Features**
   - Tokens expire after 30 days
   - Old tokens automatically cleaned up
   - Cookie is httpOnly (JavaScript can't access)
   - Token is hashed (can't be reversed)
   - Database verifies token ownership

### Cookies Used:
```php
remember_token    # Secure, httpOnly, 30 days
remember_email    # User's email, 30 days
```

### For Developers:

**Add New Product:**
```php
INSERT INTO products (name, description, category_id, price, discount_price, stock, image_url, status)
VALUES ('Product Name', 'Description', 1, 1000, 800, 50, 'image.jpg', 'active');
```

**Get User's Orders:**
```php
$user_id = $_SESSION['user_id'];
$result = $conn->query("SELECT * FROM orders WHERE user_id = $user_id");
```

**Check If User Logged In:**
```php
if ($auth->isLoggedIn()) {
    $user = $auth->getCurrentUser();
    echo $user['name'];
}
```

In `checkout.php`, line ~45:
```php
$tax = $subtotal * 0.10;  // Change 0.10 to your rate
```

### Change Free Shipping Amount:
In `checkout.php`, line ~46:
```php
$shipping = $subtotal > 5000 ? 0 : 100;  // Change 5000
```

### Change Session Timeout:
In `config.php`:
```php
define('SESSION_TIMEOUT', 3600);  // 1 hour in seconds
```

### Change Remember Me Duration:
In `config.php`:
```php
define('REMEMBER_ME_DURATION', 30 * 24 * 60 * 60);  // 30 days
```

## 🚨 Troubleshooting

### "Database Connection Failed"
- Check MySQL is running
- Check credentials in `config.php`
- Ensure `daraz_ecommerce` database exists

### "Page Not Loading"
- Check URL: `http://localhost/Lab%20Report%2005/`
- Note the `%20` for space
- Check all files are in correct folder

### "Remember Me Not Working"
- Check cookies are enabled
- Clear browser cookies
- Check database `remember_tokens` table
- Verify `httpOnly` cookies are set

### "Can't Add to Cart"
- Ensure you're logged in
- Check product has stock
- Check PHP error logs
- Clear browser cache

### "Checkout Error"
- Fill in all required fields
- Check product stock
- Verify shipping address
- Check database connection
